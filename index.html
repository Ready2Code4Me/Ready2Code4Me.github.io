Functions

Variable နဲ႔ လုပ္ေဆာင္ခ်က္ေတ တာဝန္ေတကို ခိုင္းေစလို႔မရပါဘူး။ လုပ္ေဆာင္ခ်က္ေတ တာဝန္ေတကို အလုပ္လုပ္ေစရန္ function ကို အသံုး ျပဳေပးရပါတယ္။
javascript function တစ္ခုတည္ေဆာက္ရန္ function ဆိုတဲ့ keyword ေနာက္မွာ functionName နဲ႔ parameters ေတထည္႔ေပးရပါတယ္။


Javascript Basic Function

  function doAlert(){
    alert("Alerting !!!");
  }


  doAlert();       // call function


တည္ေဆာက္ထားတဲ႔ function က ဒီအတိုင္း သတ္မွတ္လိုက္ရံုနဲ႔ အလုပ္မလုပ္ေသးပါ။ function ကို (invoke) ေခၚယူအသံုးျပဳမွသာ အလုပ္လုပ္မွာျဖစ္ပါတယ္။



Function With Parameters

function define လုပ္တဲ႔အခါမွာ ေပးတဲ႔ variable ေတကို parameter လို႔ ေခၚျပီး function call တဲ႔ အခါ ေပးရတဲ႔ value ေတကို argument လို႔ ေခၚပါတယ္။

Function with parameter 

 function add(a,b){    // a & b called parameter
  console.log (a+b);
}

    add(3,4);          // answer -> 7  
                       // 3 & 4 called arguments

Warning :: return

  return a+b;

}

Function တစ္ခုမွာ return နဲ႔ ရလဒ္ကိုျပန္ေပးျပီးရင္ သူ႔အလုပ္ျပီးပါျပီ။
return ေအာက္မွာေရးထားတဲ႔ code ေတကို ဆက္အလုပ္မလုပ္ေပးေတာ့ပါ။



Warning :: When giving arguments

Arguments ေတ ေပးတဲ႔အခါမွာ အတိအက် ေပးရပါတယ္။
Note :: Argument ပိုေပးတဲ႔အခါ ပိုတာေတကို ဒီအတိုင္းထားပါတယ္။အလုပ္မလုပ္ပါ။


Giving Arguments

 function add(a,b){
  return a+b;
}

 add(1,2,3,4);     // 3
 add(1);           // NaN


Arguments ေပးတာလိုတဲ႔အခါမွာေတာ့ Error တက္ပါတယ္။ undefined or NaN


Default Parameter

Default Parameter ေပးထားရင္ေတာ့ Arguments ေပးတာလိုရင္လဲ ျပႆနာ မရွိပါဘူး။

Function Default Parameter

 function add(a,b=1){
  return a+b;
}

 add(1,2)      // 3
 add(1)        // 2


Argument မ ေပးရင္ default parameter သတ္မွတ္ထားတဲ႔ value နဲ႔ဘဲ အလုပ္လုပ္ ေပးမွာပါ။


Rest Parameter (...paraname)

ပါလာသမ်ွ argument တန္ဖိုးအားလံုးကို လက္ခံေပးႏိုင္တဲ႔ parameter ပါ။
...paraname ဆိုျပီး အသံုးျပဳရပါတယ္။

Rest Parameter

 function add(a,b,...c){
  console.log(c);
}
  
 add(1,2,3,4,5)          // [3,4,5]

ပိုတဲ႔ argument ေတကို "c" က လက္ခံေပးထားပါတယ္။



Anonymous Function (Nameless function)

Function name ေပးစရာမလိုဘဲ variable ထဲမွာ function ကို assign လုပ္ျပီး variable က ေန function call ျပန္ေခၚတာကို nameless function လို႔လဲ ေခၚပါတယ္။


Anonymous Function

 let greet = function(){
  console.log("Hello World")
}


 greet();



IIFE ( Immediately Invoked Function Expression )

Function အလုပ္လုပ္ေအာင္ ေခၚစရာမလိုဘဲ သူ႔ဘာသာ auto ခ်က္ခ်င္းေခၚတဲ႔ function ကို IIFE function လို႔ ေခၚပါတယ္။

Javascript


(function(){
  console.log("Hello World")
})()


Function call ေခၚစရာမလိုဘဲ Console မွာ Hello World ထုတ္ေပးပါလိမ္မယ္။



Callback Function

ဥပမာ functionOne နဲ႔ functionTwo ဆိုျပီး function နွစ္ခုရွိတယ္ဆိုပါေတာ့။
functionOne ကို ေခၚတဲ႔အခါ functionTwo ကို argument အ ေနနဲ႔ ထည္႔ ေပးျပီး funtionOne ထဲမွာ functionTwo ကို ထည္႔အလုပ္လုပ္ေစတာကို Callback function လို႔ ေခၚပါတယ္။

Javascript

// *** Declaring callback function ***//

  function twice(num,fun){
   let result = fun(num)        // 6
   return result * 2            // 12
}


// *** Calling callback function ***//
 
  twice( 5 ,function(x){
  return x+1;               // 6
});
  


// The above example without callback will be same the following

  function twice(num){
   let result = function(){ return x+ 1};  // 6
   return result * 2;                      // 12
}

  twice(5);


Callback Function ၏ အားသာခ်က္

Javascript function ေတဟာ ၾကိဳေရးထားတဲ႔အတိုင္း ပံုေသမဟုတ္ေတာ့ဘဲ ေခၚယူခ်ိန္မွာ လိုအပ္သလို ေျပာင္းလဲ အလုပ္လုပ္ေစနိုင္တဲ႔ function ေတ ျဖစ္လာပါတယ္။

Example


Dyanamic callback function (example)

  function twice(num,fun){
   let result = fun(num)        
   return result * 2            

  twice(5,function(x){ 
    return x + 1;         
  });                   // return answer will be 12

  twice(5,function(x){ 
    return x * x;         
  });                   // return answer will be 50

  twice(5,function(x){ 
    return x -1;         
  });                   // return answer will be 8



Arrow Function (function အတိုေကာက္) ေရးနည္း


1. ( ) ေရွ႔က function keyword ကိုဖယ္ျပီး ( ) ေနာက္မွာ => ထည္႔ ေပးရပါတယ္။

Arrow Function

// anonymous function 

  let add = function(a,b){
   return a+b;         
}


// arrow function
  
  let add = (a,b) => {
   return a+b;
}


2. function မွာ Statement တစ္ေၾကာင္းဘဲရွိရင္
{ } မလို ၊ return မထည္႔ဘဲေရးလို႔ရပါတယ္။ auto return ျပန္ေပးပါတယ္။

Arrow Function

 let add = (a,b) => a+b        // add(3,4)


3. parameter တစ္ခုထဲ ဆိုရင္ () မထည္႔လို႔ရပါေသးတယ္။

Arrow Function

 let two = n => n*2        // two(4)


4. parameter မရွိရင္ေတာ့ () ထည္ေပးရပါမယ္။
